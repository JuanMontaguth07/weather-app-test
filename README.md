# Weather App – Prueba Técnica

Esta es una aplicación web simple para consultar el clima en tiempo real con modo día/noche, paisaje animado y una interfaz ligera.

##  Requerimientos para ejecutar la aplicación

- Navegador moderno (Chrome, Edge, Firefox u Opera)
- Conexión a internet (la app consulta una API pública)
- No necesita instalación ni dependencias externas

## Cómo ejecutar la aplicación

1. Descarga el repositorio o clónalo:
2. Abre el archivo `index.html` en tu navegador.
3. Ingresa el nombre de una ciudad y dale a **Buscar**.

## Características principales

- Modo **día/noche** con animación.
- Cielo, nubes, estrellas, meteoros, pasto y ciudad animados.
- Consulta de clima actual.
- Pronóstico básico.
- Diseño responsivo.

##  Archivos principales
- `index.html` — estructura de la aplicación
- `style.css` — estilos y animaciones
- `script.js` — lógica de la API (si aplica)
- `/img` — recursos gráficos
